# Справочная библиотек

Веб-приложение для поиска книг и управления библиотечным фондом.

## Функционал

### Пользователь
- Поиск книг по названию, автору, жанру
- Просмотр, в каких библиотеках есть книга

### Администратор
- Вход в админ-панель
- Добавление, редактирование, удаление книг и библиотек
- Привязка книг к библиотекам

## Установка и запуск

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
2. Запустите приложение:
   ```
   python app.py
   ```
3. Откройте в браузере: [http://127.0.0.1:5000/](http://127.0.0.1:5000/)

## Структура БД
- `books` (id, title, author, genre)
- `libraries` (id, name, address)
- `book_library` (book_id, library_id)
- `admin` (id, username, password)

## Примечания
- Для первого входа администратора добавьте пользователя вручную через SQLite или расширьте функционал регистрации.
